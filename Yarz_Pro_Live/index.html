Data Cleanup

⚠️ These options will delete data from your Supabase database. Use carefully.

1. Clean Financial data only
Your Product Inventory will remain untouched. Only Revenue, Transactions, Orders, Ad Spend, and Expenses will be reset. A new financial period will start today.
2. Clean Products and Inventory only
Financial data will remain intact. Only Inventory Products will be deleted.
3. Complete Factory Reset
Everything will be deleted — Products, Inventory, Orders, Finances — All. It will be like a new website.
